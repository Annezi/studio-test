# SEO Template For New Pages

Use this template whenever you add a new page.

## 1) URL and Slug
- Use human-readable URL with target query: `service-site-audit.html`
- One page = one primary intent/query cluster

## 2) Metadata Rules
- `title`: 50-65 chars, primary key in first half
- `meta description`: 130-170 chars, benefit + CTA
- `h1`: unique, close to title, natural wording
- `canonical`: self-referencing absolute URL
- `robots`: `index, follow`

## 3) Content Structure
- 1x H1
- 3-5x H2
- Optional H3 blocks for details
- 900+ words for commercial pages
- 1200+ words for blog guides
- Include synonyms: `создание сайтов`, `разработка сайтов`, `создание интернет-магазина`, `SEO-оптимизация`

## 4) Uniqueness Checklist
- No copy-paste between service pages
- Each intro paragraph must be rewritten
- Use concrete examples and niche context
- Avoid repeated sentence patterns

## 5) Internal Linking
- Link to 3-6 relevant pages
- Add links to contact and pricing pages
- Use descriptive anchor texts

## 6) Images
- At least 1 thematic image per 2-3 screen blocks
- Descriptive `alt` text with natural keywords
- Keep fallback script enabled (`assets/site.js`)

## 7) Schema
- Mandatory: `WebPage`
- For service page: `Service` + `BreadcrumbList`
- For article: `Article` + `BreadcrumbList`
- For FAQ block: `FAQPage`

## 8) Publication QA
- Validate title and description uniqueness
- Check no broken links
- Verify sitemap includes new URL
- Re-submit sitemap in Yandex Webmaster

## 9) Yandex-specific
- Track indexing/coverage in Webmaster
- Watch snippets and CTR
- Expand pages based on real queries from impressions
